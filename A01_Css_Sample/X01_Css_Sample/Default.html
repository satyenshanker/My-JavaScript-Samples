Simple CSS Example

Cascading Style Sheets (CSS) is a style sheet language used for describing the look and formatting of a document written in a markup language.

Basically we are moving the properties and styling parameters of the HTML tags into the CSS script file instead of hard coding it in the content HTML page

For Example
      <h1 color:blue>This is a heading</h1>
      can be redefined as
      <h1>This is a heading</h1>       In HTML page
     And move the property color:blue into CSS page
